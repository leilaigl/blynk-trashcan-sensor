# blynk-trashcan-sensor
Code and schematics for a Blynk-optimized trash bin level monitoring device.

## Hardware Used
- NodeMCU v1
- Half-size breadboard
- HC-SR04 Ultrasonic module
- Male-to-male MicroUSB to USB cable
- 5V Powerbank
- Jumper wires
- Phone with Blynk app
